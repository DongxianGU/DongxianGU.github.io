矩阵分析与计算回顾

回顾矩阵和分析重点知识点和内容

范数

  • 向量范数 $L_1,L_2,L_p$
  • 矩阵范数 $m1,m_\infty,F$
  • 算子范数：极大列和范数，极大行和范数，谱范数
  • 算子范数和矩阵范数的关系
  • 条件范数$cond(A)$在误差估计的应用
  • 正规矩阵，Hermite矩阵（对称举证的推广），酉矩阵（正交矩阵的推广），正定举证
  • 距离，赋范线性空间

矩阵的特征值

  • Smith标准型

  • $\lambda$-矩阵，不变因子，行列式因子，初等因子

  • Jordan标准型

  • 盖尔圆，特征值的隔离

  • 幂迭代法，逆幂迭代法 求主特征值和特征向量，以下为幂迭代法的公式

矩阵分解和广义逆

  • 三角分解，Doolittle 分解
  • Chomlesky 分解
  • QR分解
  • 满秩分解
  • 奇异值分解
  • 广义逆
  • Moore-Penrose 逆的直接解法：满秩分解，奇异值分解

矩阵的数值计算

  • Gauss 消元法，选列主元。
  • 三角分解
  • 平方根，改进平方跟
  • …
  • J迭代法，G-S迭代法，SOR迭代法
  • 极小化方法：（将矩阵计算转为泛函极值问题）最速下降法，共轭梯度法

最优化方法

  • 单纯形法
  • K-K-T点求解
  • 最速下降法，牛顿法，阻尼牛顿法，共轭梯度法
  • 罚函数法：外点罚函数，内点罚函数
  • 广义乘子法
