nmap基础教程

介绍

Nmap主机扫描工具

主要功能：

  • 主机探测
  • 端口扫描
  • 版本检测
  • 系统检测
  • 支持探测脚本的编写

应用场合

  • 通过对设备或者防火墙的探测来审计它的安全性
  • 探测目标主机所开放的端口
  • 通过识别新的服务器审计网络的安全性
  • 探测网络上的主机

端口扫描工具

借助工具，收集扫描IP提供哦你的计算机网络服务类型，从而发现弱点。

常见服务对应端口

服务 端口号

HTTPS 443

MySQL 数据库sever 3306
